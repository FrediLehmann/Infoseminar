\section{\index{Jitsi}{Jitsi}}
Well known implementations of \index{WebRTC}{WebRTC} are the free and open source software solutions from \index{Jitsi}{Jitsi}. \index{Jitsi}{Jitsi} provides multiple products for video conferencing, for example the \index{Jitsi Videobridge}{Jitsi Videobridge} which is a server application used to build scalable multiparty video applications or \index{Jisti Meet}{Jisti Meet} which is a complete video conferencing solution.
